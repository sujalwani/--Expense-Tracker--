# 💰 Expense Tracker

A modern and responsive **Expense Tracker Web Application** built using **HTML, CSS, and JavaScript**. This application helps users track their income and expenses, calculate their current balance, and store transaction history using the browser's Local Storage.

## 🚀 Features

* Add income and expense transactions
* Automatic balance calculation
* Separate income and expense summaries
* Delete transactions instantly
* Data persistence using Local Storage
* Responsive design for mobile, tablet, and desktop devices
* Smooth animations and modern UI design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Google Fonts (Poppins)

## 💡 How It Works

* Enter a transaction description.
* Enter a positive amount for income.
* Enter a negative amount for expenses.
* Click **Add Transaction**.
* The balance, income, and expense totals update automatically.
* All transactions are saved in Local Storage, so data remains available even after refreshing the page.
